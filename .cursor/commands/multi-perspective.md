# /multi-perspective — Three-Angle Review

## Steps

1. Frame the proposal in one paragraph.
2. Analyze it as Optimizer.
3. Analyze it as Critic.
4. Analyze it as Devil's Advocate.
5. Synthesize the final call and note accepted risks.

## Output

Use this format:

```markdown
## 🎭 Multi-Perspective Review

### Proposal
<one paragraph>

### 🟢 Optimizer
- <bullet>

### 🔴 Critic
- <bullet>

### 🟣 Devil's Advocate
- <bullet>

### Synthesis (Final Call)
**Decision:** <chosen path>
**Why this won:** <paragraph>
**Accepted risks:**
- <risk>
**Confidence:** <high | medium | low>
```
