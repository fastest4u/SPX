# /session-end — Session Closeout

## Steps

1. Summarize the work completed this session.
2. List files touched or notes updated.
3. Note decisions, risks, and follow-ups.
4. Write or update a session log if the work was meaningful.
5. If memory or workflow files changed, run `npm run memory:verify` and mention the result.
6. If the session changed durable truth, update the relevant memory notes before ending.

## Output

Use a compact closeout with:
- TL;DR
- What changed
- Verification status
- Open follow-ups
